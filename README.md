# DSA Journey 🚀

This is my personal repo for learning Data Structures and Algorithms as a Second-year CS student. I'm learning it in both **Java** and **C++** side by side, mostly because my college covers one and I want to be comfortable in the other too.

I'm not an expert — this repo is basically my rough notebook turned into code. Expect messy commits, half-finished solutions, and comments to my future self.

## Why I'm doing this

Everyone says DSA is the backbone of coding interviews and problem solving, so I figured the earliest I start, the less I'll panic in 3rd year.

## How I'm learning

- College class notes and lectures
- Abdul Bari's DSA playlist on YouTube (genuinely the GOAT for building intuition)
- Random YouTube videos when a topic doesn't click the first time
- AI tools (like ChatGPT/Claude) when I get stuck or want a concept explained differently — I use them to understand, not to copy-paste solutions
